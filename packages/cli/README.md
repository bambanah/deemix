```
Usage: deemix [options] <url>

A CLI wrapper for deemix

Arguments:
  url                   The URL of the track or playlist

Options:
  -V, --version         output the version number
  -p, --path <path>     Downloads in the given folder
  -b, --bitrate <type>  Overrides the default bitrate selected
  --portable            Creates the config folder in the same directory where the script is launched
  -h, --help            display help for command
```
